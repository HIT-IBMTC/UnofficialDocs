浏览 API
*****************

快速查看API的方法：

#. 将 ``Packages\Default`` (**Preferences | Browse Packages…**) 添加到你的项目中.
#. 按下组合键 ``CTRL + SHIFT + F``
#. 在 **In Files:** 栏中输入 ``*.py``
#. 检查 ``Use Buffer`` 选项
#. 搜索 API 名字
#. 按 ``F4``
#. 学习相关的源代码
